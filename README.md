# marvel-clicky

<h1> A Click Game with Marvel Logos</h1>
This is a clicky game where you click the tweleve Marvel Heroes logos without clicking the same one twice.

<h3>This was built using:</h3>
<ul>
    <li>React-DOM</li>
    <li>React-Scripts</li>
    <li>React</li>
    <li>Node</li>
    <li>React bootstrap</li>
</ul>
